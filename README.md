# isenseflu-app-react

Frontend component of i-sense flu.

## Requirements

- Node >= 6
- npm >= 5.2

## Building

Pass the `REACT_APP_API_HOST` variable pointing to the backend component that serves the data for 
Frontend app.

```
REACT_APP_API_HOST=[hostname_of_api_endpoint:port] npm run build
serve -s build
```
